This is the first post on the site and a useful template for the next one.

Jekyll posts live in `_posts/` and follow the filename pattern:

`YEAR-MONTH-DAY-title.MARKUP`

Each post starts with front matter that sets the title, date, and optional
categories. After that, the rest of the file is just Markdown.

For local development, run:

```bash
bundle exec jekyll serve
```

Then open `http://127.0.0.1:4000` and start writing.

The current theme is custom and lightweight, so the main files to edit are:

- `_config.yml` for site metadata
- `index.markdown` for the homepage intro
- `about.markdown` for your bio
- `assets/css/style.css` for the look and feel
